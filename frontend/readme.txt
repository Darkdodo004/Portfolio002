Frontend

The frontend is basically divided into 5 subfiles which is home,skills,about,services and contact. There is a nav which is sticky .
All the jsx files are in the components folder in the src folder in which the header file is the navbar linked with the main file, The Home file contains the main body. 
There is a Styles folder in the src folder which contains all the scss file with mediaqueires.
The Assets folder in the src folder contains all the images for the project.
Animations are also added in this project following with crousel part to enhance the look and to be mmore attractive .
